# Online Book Store Website

This is a **front-end online book store website** created as a personal study project.

## Features

- Landing page with hero section and call-to-action
- Navigation bar with links to:
  - Home
  - About Us
  - Books (categories for Adults, Teens, and Children)
  - Blog
  - Services
  - Free book / booking
  - Contact page
- Sections for:
  - Services
  - Featured books with images, prices, and external links
  - Customer reviews
  - Footer with links and newsletter input

## Technology Used

- **HTML5** for the structure
- **CSS3** for layout, styling, and hover effects
- External icons (Font Awesome CDN)
- Responsive layout basics using flexible boxes

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Navigate through the different pages using the navigation bar.

This project was created to practice **front-end web development**, page structure, and styling for an online store concept.
